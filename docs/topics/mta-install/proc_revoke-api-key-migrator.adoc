= Revoke API keys as a migrator

[role="_abstract"]
As a migrator, you can revoke an application programming interface (API) key before its lifespan period in {ProductFirstRef} web console.

.Prerequisites

* You have installed {ProductShortName} {ProductVersion} in a Red{nbsp}Hat OpenShift cluster.

.Procedure

. Log in to the {ProductShortName} web console.
. In the *Migration* mode, in the upper-right corner, click your username.
. Click *Tokens*.
. On the *Tokens* page, click the options icon.
. Click *Revoke*.

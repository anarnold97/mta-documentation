= Revoke API keys as an administrator

[role="_abstract"]
As an administrator, you can revoke an application programming interface (API) key before its lifespan period in the migration toolkit for application (MTA) web console.

.Prerequisites

You installed {ProductShortName} {ProductVersion} in a Red{nbsp}Hat OpenShift cluster.

.Procedure

. Log in to the {ProductShortName} web console.
. In the *Administration* mode, click *User Management*.
. Click *Tokens*.
. On the *Tokens* page, click the options icon.
. Click *Revoke*.

= OpenLDAP and Active Directory configurations

[role="_abstract"]
OpenLDAP and Active Directory (AD) implementations require different configurations for specific fields.

.OpenLDAP and Active Directory configurations
[cols="1,1,1",options="header"]
|====
|Fields
|Active Directory
|OpenLDAP

|Username attribute
|`sAMAccountName`
|`uid`

|Group objectClass
|`group`
|Generic wildcard (`*`)

|`memberOf` support
|Usually present
|May not be present

|`userFilter`
|`(sAMAccountName=${login})`
|`(uid=${uid})`

|`groupFilter`
|`(&(objectClass=group)(member=${dn}))`
|`(&(objectClass=*)(member=${uid}))`
|====
